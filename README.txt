= Features =

- Seamless integration of the CleanDesk AI chat widget.
- Customizable settings to manage the CleanDesk AI app ID, app secret, and widget enable/disable functionality.
- Display the widget on your website pages.

== Installation ==

1. Upload the `cleandesk-ai-widget` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Create an account in https://www.cleandesk.ai and obtain the App ID and App Secret from Settings > Credentials.
4. Configure the plugin settings under 'CleanDesk AI' in the WordPress admin menu.

== Frequently Asked Questions ==

= How do I obtain CleanDesk AI credentials? =

To use the CleanDesk AI Widget, you need to sign up on the CleanDesk platform to obtain the App ID and App Secret.

= Can I customize the appearance of the chat widget? =

Currently, the appearance customization options are limited. You can, however, modify the script directly if you need advanced customization.
